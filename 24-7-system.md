# 24/7 自動化開發系統

## 運作方式

```
1. 你提出需求 (Telegram/Email)
2. Main Agent 自動接收並拆解任務
3. 子代理並行執行
4. 定時回報進度
5. 完成後自動通知
```

## 設定

### 1. Cron 排程 (自動觸發任務)
```bash
# 每日早上 9 點檢查進度
cron: "0 9 * * *"

# 每小時健康檢查
cron: "0 * * * *"
```

### 2. 子代理設定
```json
{
  "subagents": {
    "maxConcurrent": 8,
    "runTimeoutSeconds": 7200,
    "thread": true,
    "mode": "session"
  }
}
```

### 3. Heartbeat 設定 (進度監控)
```
每 30 分鐘檢查:
- 子代理狀態
- 任務進度
- 是否有錯誤
```

## 使用範例

### 範例 1: 陳列設計
```
你：「幫我設計聖誕節門市陳列」

系統自動：
1. 研究聖誕節趨勢
2. 生成陳列概念圖
3. 製作提案簡報
4. 寄送給你
```

### 範例 2: 活動規劃
```
你：「幫我規劃春酒活動」

系統自動：
1. 研究活動類型
2. 製作流程表
3. 生成海報
4. 輸出完整企劃
```

### 範例 3: 數據報告
```
你：「這個月銷售報告」

系統自動：
1. 收集數據
2. 分析趨勢
3. 生成圖表
4. 寄送報告
```

## 通知設定

| 觸發條件 | 通知方式 |
|----------|----------|
| 任務完成 | Telegram |
| 遇到錯誤 | Telegram + Email |
| 進度更新 | Telegram (可選) |
| 每日摘要 | Email |

## 維護

- 每日記憶體備份
- 每週回顧優化
- 定時檢查 API 配額
